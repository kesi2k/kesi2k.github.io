Leagues

This project uses Google plus for user registration and authentication. Authenticated users can perform create, read, delete and update actions on data that they have created. The data is in the form of a league, that contains teams and these teams contain rosters. Server side code uses Python. This project is hosted on heroku.

User authenticated blog

This project allows registered users to perform create, read, delete and update actions on their posts. They can also comment and like other posts. Server side code written in Python. This project is hosted by Google.
