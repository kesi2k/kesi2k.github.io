Resume

Leagues

This project uses Google plus for user registration and authentication. Authenticated users can perform create, read, delete and update actions on data that they have created. The data is in the form of a league, that contains teams and these teams contain rosters. Server side code uses Python. This project is hosted on heroku.

User authenticated blog

This project allows registered users to perform create, read, delete and update actions on their posts. They can also comment and like other posts. Server side code written in Python. This project is hosted by Google.

Neighborhood Map

A map containing areas of interest. The interest models are controlled with Knockout.js. AJAX requests to wikipedia for information. Async requests to google maps API for map locations.

Simon memory game

A browser replication of the memory game Simon.

Simon memory game

The webpage makes an AJAX request to wikipedias servers with the users' search. It receives a response in JSON that is used to build the webpage.

Simon memory game

A timer that keeps track of work and break duration. The work timer ranges from 5 minutes to 60 minutes. The break timer ranges from 1 minutes to 15 minutes.

Browser tic-tac-toe

This tic-tac-toe game allows the user to play against the browser. Currently, the algorithm only allows Javascript to play a random position.

Weather App

This weather API app makes a JSON request for the browsers location.Subsequently, the location is used in a JSON request to a weather database.

Laravel photo blog

This is a full stack project in Laravel. Users have the ability to upload photos. Comments can be made under posts and photos. Server side code in Php using the Laravel library. Deployed and hosted on a personal vultr server. I undertook this project to help acquaint myself with Laravel as it is the framework we are using for our Saas platform.

React youtube API

This is a front-end project built using the react Library and Youtube's API.

React Redux

This is a front-end project built using the react and redux libraries. It allows users to perform C.R.U.D. actions on posts and comments. Users can like posts and order posts by likes and the date. Please note comments section will take a while to load while waiting on heroku servers get started.
